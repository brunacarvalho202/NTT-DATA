Introdução ao HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web. "Hipertexto" refere-se aos links que conectam páginas da Web entre si, seja dentro de um único site ou entre sites. Links são um aspecto fundamental da web. Ao carregar conteúdo na Internet e vinculá-lo a páginas criadas por outras pessoas, você se torna um participante ativo na world wide web.

Estrutura básica de HTML

Por padrão do W3C os arquivos de páginas HTML ficam no arquivo de projeto princiapl chamado "index.html" que têm esse nome por padrão e boas práticas, mas pode ser modificado. O HTML é estruturadas em 3 tags principais: html, head e body.

  1. A tag de html A tag de html faz com que o navegador entenda que tudo dentro do arquivo que ee está interpretando é html.
  2. A tah head recebe informações gerais do documento, mais conhecidos como metadados, como por ex o titulo que fica na aba do navegador atraves da tag title.
  3. A tag body

Obs: O World Wide Web Consortium (W3C) é a principal organização de padronização da World Wide Web. Consiste em um consórcio internacional com 450 membros, agrega empresas, órgãos governamentais e organizações independentes com a finalidade de estabelecer padrões para a criação e a interpretação de conteúdos para a Web.

Tags em HTML

As tags são elementos que têm uma importância significativa na estrutura do seu site, visto que é ela quem informa ao navegador que tipo de estrutura vai ser usada para a informação que vc quer construir no site (ela estrutura sua informação no documento), podendo indicar, por ex, que é um titulo, paragrafo, imagem, link e outros recursos de estrutura html. Existem duas formas especificas para essa tag ser digitada no código: usando seu nome dentro de sinal maior e menor (< NomeDaTag >). Além disso, toda tag precisa ser aberta/inicializada e fechada com uma barra dentro do simbolo (< NomeDaTag />). Algumas tags específicas se autofecham, pq possuem apenas uma sinalização de inicialização já que não precisam receber informação dentro delas. Dois exemplos delas são < img /> < input/> Obs: outras tags podem ser usadas dentro de uma.

Um exemplo de tag em HTML é a < small > < / small > que deixa o texto, exatamente assim, com a fonte menor que a padrão.

Outros exemplos interessantes de tag HTML é < sup > < / sup > que deixa o texto no índice de outro como: X2 e < sub > < / sub > que faz com que esse íncide fique abaixo do texto como: X2

Atributos para tags em HTML

Os atributos são basicamente recursos que configuram as tags. Existem as tagas mais genéricas, que todas as tagas possuem e, também, têm os que são espcíficos para cada tag. A aplicação desse atributo dentro do código é colocando sua estrutura dentro do nome da tag. Ex: < input type= "text" />

Os 3 atributos mais genéricos são:

  • atributo "id"

    serve como um identificador para aquele elemento

  • atributo "style"

    é um atributo mais voltado para o css,que é uma forma de formatar o html, serve para colocar os comandos css direto no elemento.

  • atributo "class"

    É um outro atributo voltado para o css, no qual vc moonta uma classe e adiciona todas as caracteristicas de formtação que vc deseja. Tudo que estiver nessa classe, quando vc direciona o elemnto para ela, vai ser padronizado como estar lá.

Exemplos de atributos específicos:

  • atributo "type" para tag "input" O type pode receber alguns tipos de informações, como texto, numeros e até cores.
    • type de texto: cria um campo no input para receber apenas texto

      estrutura: < type = "number" />

    • type de numeros: cria um campo de input para receber números, inclusive com umas setas para aumentar ou diminiui esse número.

      estrutura: < type = "text" />

    • type de cores: cria um campo de input com a opção de seletor de cores para estar nesse campo do input.

      estrutura: < type = "color" />

  • atributo "src" e "alt" da tag img a tag img recebe imagens para precisa do atributo "src" para direcionar onde está essa imagem e qual é, podendo ser um link de internet ou o caminho do arquivo e pasta onde se encontra essa imagem. Resumidamente, o src recebe o endereço dessa imagem. Já o atributo alt recebe a descrição dessa imagem inserida.

    estrutura: < img src="endereço de imagem " alt=" descrição dessa imagem ">

    • observação: caso a img fique grande, outro atributo disponivel é o widht que formata a largura.

    estrutura: < img src="endereço" alt="descrição" width="numero da largura">

Lidando com textos em HTML

Algumas tags nos fornecem a capacidade de manipular e estruturar os textos em html. Veremos algumas:

  • Tag "h" são as tags para títulos. Possui subdivisões de h1 até h6, onde cada tag consecutiva se comporta como o subtitulo da anterior. Vale lembrar que o navegador entende que, por ex, h2 é subtitulo de h1 e por padrão da um espaçamento entre eles, podendo, também, ser alterado via css

    titulo 1

    titulo 2

    titulo 3

    titulo 4

    titulo 5
    titulo 6
  • Tag "blockquote" serve para padronizar uma citação despredendo esse texto do resto e adicionando um espaçamento e margem.
    citação
  • Tag "u" de underline, sublinha um texto.
  • Tag "i" deixa o texto em itálico.
  • Tag "mark" marca o texto com a cor padrão amarela, mas pode ser mudada com css

Listas ordenadas e não ordenadas em HTML

O HTML possui duas formas de listas textos, ordenadamente (tag ol) ou desordenadamente (tag ul). Cada forma com sua tag específica:

Estruturas:

  • lista ordenada: < ol> adicionar as tags li para cada item < /ol>
  • lista não ordenada: < ul> adicionar as tags li para cada item < /ul>

A lista ordenada organiza seus itens(tag li) por numerações e por isso necessita das suas tags filhas (para cada item)

    lista ordenada:
  1. item 1
  2. item 2
  3. item 3

A lista não ordenada não enumera seus itens, apenas os separa com espaçamentos e marcadores de bolinhas (podem ser mudados com css)

    lista não ordenada:
  • item 1
  • item 2
  • item 3
